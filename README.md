# JDBC CRUD Demo Project

## Project Overview
This project demonstrates basic **CRUD operations (Create, Read, Update, Delete)** using **JDBC with PostgreSQL** in Java.  
It is a **console-based application** intended to help understand core JDBC concepts without using any framework.

---

## Technologies Used
- Java (JDK 21)
- JDBC
- PostgreSQL
- Maven
- Eclipse IDE
- Git & GitHub
